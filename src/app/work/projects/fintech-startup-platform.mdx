## Overview

Co-founded Eco AI.ly, a startup uniting artificial intelligence with environmental sustainability. As Machine Learning and Full-Stack Developer, I led development of "GAIA" (Green AI Assistant), an AI-powered platform that monitors grid carbon intensity and forecasts renewable energy trends. The platform helps companies optimize energy usage and reduce carbon footprint through data-driven insights.

## Key Features

- **Carbon Intensity Forecasting**: Built LSTM neural networks that forecast carbon intensity and renewable energy output 24 hours ahead with ~91% accuracy.
- **Real-time Energy Monitoring**: Developed FastAPI microservice that serves live predictions and monitors real-time power grid data.
- **Interactive Dashboards**: Created Next.js/Streamlit dual-frontend for interactive dashboards and automated PDF reporting with comprehensive analytics.
- **Cloud Deployment**: Deployed on cloud with containerization for scalability, focusing on Portugal energy grid with actionable insights.
- **Sustainability Insights**: Provides actionable insights for timing energy usage to periods of low carbon intensity, enabling sustainable energy planning.

## Technologies Used

- **Python**: Core development language with TensorFlow for LSTM neural network development
- **TensorFlow**: Deep learning framework for time-series forecasting and energy prediction models
- **FastAPI**: High-performance microservice for serving live predictions and API endpoints
- **Docker**: Containerization for scalable deployment and cloud infrastructure
- **Next.js**: Modern React framework for interactive web dashboards
- **Tailwind CSS**: Utility-first CSS framework for responsive and modern UI design
- **ElectricityMaps API**: Real-time energy grid data integration for carbon intensity monitoring

## Technical Achievements

- **Prediction Accuracy**: Achieved 90.9% accuracy in 24-hour carbon intensity and renewable energy forecasting
- **Real-time Processing**: Built scalable system processing live energy grid data with sub-second response times

- **Data Pipeline**: Implemented robust data processing pipeline handling real-time energy grid data feeds
