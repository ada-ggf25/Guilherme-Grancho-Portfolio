## Overview

Co-founded Tap2Net, an entrepreneurial project designed to help businesses enhance their online reviews through seamless customer engagement. Tap2Net provided physical NFC-enabled cards that, when tapped by a customer's smartphone, redirect to a personalized review page for the business. This innovative solution addressed the critical need for small businesses to build and maintain strong online reputations.

## Key Features

- **NFC-Enabled Smart Cards**: Developed physical NFC cards that customers can tap with their smartphones to instantly access business review pages.
- **Personalized Review Pages**: Created customized review interfaces for each business, streamlining the customer feedback process.
- **Web Application Platform**: Built comprehensive web app for business owners to manage their review pages, track feedback, and monitor online reputation.
- **Hardware Integration**: Managed NFC tag programming and hardware integration for seamless customer experience.
- **Customer Analytics**: Implemented tracking and analytics to help businesses understand customer engagement and review patterns.

## Technologies Used

- **JavaScript**: Core development language for web application and frontend development
- **Firebase**: Backend-as-a-Service for user authentication, database, and hosting
- **NFC Technology**: Near Field Communication tags for seamless smartphone integration
- **WordPress**: Landing page development and content management
